## 函数命名

javascript 的命名有以下几种

```javascript
function fn() {
  return "fn";
}
var fn1 = function fn1() {
  return "fn1";
};
//匿名
var fn2 = function() {
  return "fn1";
};
//lambda
var fn3 = () => {
  return "fn3";
};
//直接返回值的lambda，相当于()=>{return 123}
var fn4 = () => 123;
```
